[Nest](https://github.com/nestjs/nest) framework TypeScript boilerplate with Postgres database and drizzle ORM.


## Features
- API documentation with swagger
- JWT Cookie
- Typescript
- Validation & Serialization

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## CLI Commamds

```bash
# Add new resource
$ nest g resource users --no-spec

```
